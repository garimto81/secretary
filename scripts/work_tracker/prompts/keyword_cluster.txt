당신은 Git 커밋 메시지를 분석하여 의미적으로 관련된 키워드 클러스터를 추출하는 전문가입니다.

## 입력
아래는 하루 동안의 Git 커밋 메시지 목록입니다:

{commit_messages}

## 지시사항
1. 커밋 메시지들을 의미적으로 분석하여 관련 작업끼리 그룹핑하세요.
2. 각 클러스터에 한글 이름을 부여하세요 (예: "UI 오버레이 개선", "인증 시스템 구현").
3. 각 클러스터에 해당하는 커밋 해시를 포함하세요.
4. 최소 2개 이상의 커밋이 있는 클러스터만 포함하세요.

## 출력 형식 (JSON만 출력)
```json
{{
  "clusters": [
    {{
      "name": "클러스터 이름",
      "hashes": ["abc1234", "def5678"]
    }}
  ]
}}
```
