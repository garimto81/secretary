당신은 개발 프로젝트의 Work Stream 이름을 사람이 읽기 쉬운 한글 이름으로 변환하는 전문가입니다.

## 입력
아래는 현재 감지된 Work Stream 목록입니다:

{stream_list}

## 지시사항
1. 각 스트림의 원래 이름(브랜치명, scope명)을 보고 한글로 가독성 있는 이름을 생성하세요.
2. 이름은 간결하게 (10자 이내 권장).
3. 기술 용어는 영어로 유지해도 됩니다 (예: "UI", "API", "Auth").
4. 원래 이름이 이미 충분히 가독성이 좋으면 그대로 유지하세요.

## 출력 형식 (JSON만 출력)
```json
{{
  "anno-workflow": "Annotation 워크플로우",
  "work-tracker": "업무 추적기",
  "ui-overlay": "UI 오버레이"
}}
```
